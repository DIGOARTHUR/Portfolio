## ✨ Como usar

- ### **Pré-requisitos**

  - É **necessário** possuir o **[Node.js](https://nodejs.org/en/)** instalado na máquina

1. Faça um clone :

```sh
  $ git clone https://github.com/thalytabdn/Portfolio
```

2. Executando a Aplicação:

```sh

  # Iniciar a aplicação web 
  $ npm run dev

 ```
